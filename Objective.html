OBJECTIVES

  • Develop an Efficient Fault Detection System: Implement a condition monitoring technique to detect bearing faults in an induction motor.
  • Utilize Advanced Signal Processing Techniques: Apply Wavelet Transform for signal analysis to extract relevant fault-related features.
  • Improve Fault Classification Accuracy: Use Support Vector Machine (SVM) as a classifier to distinguish between healthy and faulty motor conditions.
  • Create a Robust and Scalable System: Develop a fault detection method that can be extended to other types of motor faults
