Problem Statement

Motors are widely used in electric vehicles (EVs), household appliances, and industrial machinery. These motors are prone to faults due to mechanical wear, overheating, insulation failure, and unbalanced loads. If not detected early, these faults can lead to equipment damage, increased maintenance costs, and unexpected breakdowns.

This project aims to develop an AI-based fault detection system for a single-phase induction motor using vibration and sound data. The system will:

  • Collect real-time vibration and sound data using sensors connected to an Arduino microcontroller.
  • Use machine learning (Random Forest Classifier) to analyze data and detect motor faults.
  • Simulate the system using MATLAB & Simulink for testing and evaluation.
