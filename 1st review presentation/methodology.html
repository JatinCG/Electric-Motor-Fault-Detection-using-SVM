Methodology

RFC, is a supervised learning algorithm. It can be used both for classification. A forest is comprised of trees.The more trees it has, the more robust a forest is. RFC creates decision trees on randomly selected data samples, gets a prediction from each tree, and selects the best solution by means of voting. Thus, it works in four steps.

  • Select random samples from a dataset.
  • Construct a decision tree for each sample.
  • Perform a vote for each predicted result.
  • Select the most voted prediction.
