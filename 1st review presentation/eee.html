Introduction

Motor faults, such as bearing failures, stator winding faults, and rotor faults, can lead to costly downtime and safety hazards. Early detection is crucial for preventing catastrophic failures and ensuring smooth operations.Traditional methods for motor fault detection often rely on human inspection , which can be time-consuming, subjective, and prone to errors. Artificial intelligence (AI) offers a promising solution for more efficient and accurate motor fault detection.

Gap 1: Most AI models focus only on one type of data.

  • Studies like IoT-based sound analysis and vibration-only machine learning models have been explored.
  • However, combining vibration and sound data can improve accuracy, but limited research has been done on this.

Gap 2: Limited Simulink-based fault detection models.

  • Many studies focus on MATLAB programming only, but Simulink-based real-time motor fault simulation is still underdeveloped..
  • This project will integrate Simulink for better visualization and simulation of motor faults.

Problem Statement

Motors are widely used in electric vehicles (EVs), household appliances, and industrial machinery. These motors are prone to faults due to mechanical wear, overheating, insulation failure, and unbalanced loads. If not detected early, these faults can lead to equipment damage, increased maintenance costs, and unexpected breakdowns.

This project aims to develop an AI-based fault detection system for a single-phase induction motor using vibration and sound data. The system will:

  • Collect real-time vibration and sound data using sensors connected to an Arduino microcontroller.
  • Use machine learning (Random Forest Classifier) to analyze data and detect motor faults.
  • Simulate the system using MATLAB & Simulink for testing and evaluation.

Methodology

Overview of AI Random Forest Classifier (RFC) Technique

RFC, is a supervised learning algorithm. It can be used both for classification. A forest is comprised of trees. The more trees it has, the more robust a forest is. RFC creates decision trees on randomly selected data samples, gets a prediction from each tree, and selects the best solution by means of voting. Thus, it works in four steps.

  • Select random samples from a given dataset.
  • Construct a decision tree for each sample and get a prediction result from each decision tree.
  • Perform a vote for each predicted result.
  • Select the prediction result with the most votes as the final prediction.
