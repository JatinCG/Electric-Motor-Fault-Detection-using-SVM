Literature Review

Title Year Methodology
Sensor Fault Detection using Machine Learning 2021 This study applied machine learning techniques like SVM, Decision Trees, and Neural Networks to detect sensor faults in automobile drive systems. It highlighted the importance of feature selection and demonstrated improved fault detection accuracy over traditional methods.
Electrical Motor Fault Detection System using AI’s Random Forest Classifier Technique 2023 This study focused on detecting electric motor faults using vibration and sound analysis with a Random Forest classifier. It demonstrated high accuracy in fault classification and proved Random Forest’s efficiency in handling complex fault patterns.
Fault Analysis and Predictive Maintenance of Induction Motor Using Machine Learning 2018 This study explored predictive maintenance of induction motors using machine learning models such as Decision Trees and Neural Networks. It emphasized early fault detection through vibration and current signal analysis, reducing downtime and improving system reliability.
